# Banking Fraud & Transaction Analytics Dashboard

## Overview
This project presents an interactive Power BI dashboard analyzing banking transaction data, focusing on performance trends and fraud risk detection.

## Objectives
- Monitor transaction activity and performance
- Identify high-risk and suspicious transactions
- Provide insights into customer behavior and transaction patterns

## Tools Used
- Power BI
- Data Modeling
- DAX (Measures & KPIs)

## Key Features
- KPI Metrics (Total Transactions, Amount, Deposits, Withdrawals, High Risk)
- Interactive filters (Location, Category, Channel, Date)
- Transaction trends over time
- Spending analysis by category
- Fraud risk distribution
- High-risk transaction monitoring table
